Project: Slide Builder

Python Google ADK Development Kit

Generate full PowerPoint presentations using a Multi-Agent AI system (17 agents).

  • Built a 17-agent AI pipeline (Google ADK) to generate full PowerPoint decks from a single prompt
  • Designed an async task-based system to process slide sections in parallel
  • Implemented clean multi-layer architecture (Domain → Service → Repository)
  • Delivered end-to-end automation from idea → structured slides → PPTX output

Problem

Creating high-quality presentations is time-consuming and requires multiple skills: research, structuring, writing, and visual design.

Solution

Slide Builder automates the entire workflow using a Multi-Agent AI pipeline, where specialized agents handle research, content generation, and slide design.

Built with an async task-based architecture, the system generates multiple sections in parallel and compiles them into a production-ready PowerPoint file.

Impact

  • Reduced presentation creation time from hours to minutes via end-to-end automation
  • Increased throughput by enabling parallel section generation with an async task pipeline
  • Maintained consistency and quality through a multi-agent architecture with specialized roles
  • Designed a scalable system capable of handling large presentations with concurrent processing
  • Built a modular architecture that allows easy extension and iterative improvement of agents and workflows

My Role

Solo project — responsible for end-to-end design and implementation:

  • Designed system architecture and multi-agent workflow
  • Built async backend and task orchestration system
  • Implemented AI pipeline and PowerPoint generation
